## Sección 5 — System Usability Scale (SUS)

> Estas 10 frases son la **escala estándar de usabilidad** que se usa en la industria. Para cada una, indicá cuánto estás de acuerdo, de 1 (muy en desacuerdo) a 5 (muy de acuerdo). Pensá en tu experiencia general con la app.

*Tipo: matriz 1-5 para las 10 preguntas. NO renombrarlas — el cálculo del score depende de que estén en este orden y con esta polaridad alternada.*

| # | Frase | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|---|
| S1 | Me gustaría usar GelatoLab seguido. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S2 | GelatoLab me pareció innecesariamente complicada. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S3 | GelatoLab fue fácil de usar. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S4 | Necesitaría ayuda técnica para poder usar GelatoLab. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S5 | Las distintas funciones de GelatoLab estaban bien integradas entre sí. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S6 | Hay demasiada inconsistencia en GelatoLab. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S7 | Imagino que la mayoría aprendería a usar GelatoLab muy rápido. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S8 | GelatoLab fue muy incómoda de usar. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S9 | Me sentí confiado/a usando GelatoLab. | ☐ | ☐ | ☐ | ☐ | ☐ |
| S10 | Tuve que aprender muchas cosas antes de poder empezar a usar GelatoLab. | ☐ | ☐ | ☐ | ☐ | ☐ |

> *Notas para el cálculo SUS:*
> - *Impares (S1, S3, S5, S7, S9): score = (respuesta − 1)*
> - *Pares (S2, S4, S6, S8, S10): score = (5 − respuesta)*
> - *Suma los 10 valores y multiplicá por 2.5. Resultado entre 0 y 100.*
> - *Benchmark: <50 inaceptable, 50-68 OK pero hay que iterar, 68-80 bueno, >80 excelente. Promedio SaaS B2B: 68.*

## Apéndice A — Cómo armar esto en Tally.so en 30 minutos

1. Ir a [tally.so](https://tally.so) → **Sign up** (gratis, sin tarjeta).
2. **Create new form** → empezar de cero.
3. Para cada pregunta de arriba, agregar el bloque correspondiente:
   - **P1, P2, P3, P5, P6, P8, P9, P10, P11, P12, P13, P16, P19, P20, P21, P27** → Multiple choice
   - **P4, P14** → Checkboxes
   - **P15, P17** → Linear scale (1-5)
   - **S1-S10** → Matrix (escala)
   - **P18** → Linear scale (0-10) o NPS dedicated block
   - **P7, P12.b, P13.b, P15.b, P16.b, P17.b, P18.b, P22, P23, P24, P25** → Long answer
   - **P26** → Email
4. **Logic** → para preguntas condicionales (P8.b, P15.b, P21, etc.) configurar el "show if".
5. **Settings → Integrations** → Google Sheets para que las respuestas se exporten en vivo.
6. **Share → Get link** → ese link se lo das al maestro para que lo distribuya a sus alumnos.
